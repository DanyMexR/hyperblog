Parrafo de ejemplo para explicar lo que se puede hacer con git

Git es una herramienta super util, para llevar acabo todo el control de tu codigo, como por ejemplo, si 2 o mas personas estan trabajando en el mismo proyecto, cada persona podria trabajar en su ramificacion de git, sin afectar la rama principal, para cambiar de ramificacion se hace con un "checkout" y para fusionar las ramificaciones se hacen con un "merge"
